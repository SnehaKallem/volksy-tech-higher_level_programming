This doctest checks a function thats verify addition of two integers

Add two integers

>>>add_integer(1, 2)
3
>>>add_integer(100, -2)
98
>>>add_integer("2")
100
>>>add_integer(100.3, -2)
98

Return type must be int
>>>add_integer(3.5,3.5)
7

Error must be raised
>>>add_integer(4,"school")
TypeError:a must be an integer
>>>add_integer(None)
TypeError:a must be an integer
>>>add_integer(float(nan))
TypeError:a must be an integer
